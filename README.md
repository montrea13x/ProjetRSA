dotnet run
Affiche les commandes à jour

dotnet run generate  2048
Génère une nouvelle paire de clés RSA en remplaçant 2048 par la taille souhaitée (x = 2048, 3072 ou 4096)

dotnet run read
Lit et affiche les clés RSA

dotnet run encrypt
Chiffre un message avec la clé publique

dotnet run encryptfile
Chiffre un file avec la clé publique
